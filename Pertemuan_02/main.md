program Menghtiung Volume Lemari
   Program ini dapat menghitung volume lemari dengan panjang, lebar, dan tinggi yang       diinputkan oleh user.

kamus/deklarasi
    panjang, lebar, tinggi: meter
    volume: meter kubik

algoritma
    panjang <- Masukkan panjang dari user
    lebar <- Masukkan lebar dari user
    tinggi <- Masukkan tinggi dari user

    volume <- panjang * lebar * tinggi

    Tampilkan volume
end program
